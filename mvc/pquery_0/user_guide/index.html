Welcome to PQuery

PQuery stands for PHP and JQuery.PQuery is a set of Open Source PHP Classes which help you build more interactive Web Application / websites with minimal effort.PQuery help you build better applications by providing easy to use API which siplifies tasks involving Ajax , Effects , DOM etc. At its core PQuery is a JQuery javascript helper class for PHP.

To quickly start using PQuery ,read the Getting Started page.

Who is PQuery for ?

PQuery is for people who wish to build Rich Internet Applications(RIA) and want the client side scripting and effects also to be programmable though server side PHP.
